# 🚀 SaaS Product – AI YouTube SEO Tool

A full-stack SaaS web application that helps users generate YouTube SEO content like titles, descriptions, and tags using AI.

---

## 🔥 Features

* 🎯 AI-powered YouTube SEO generator
* 🔍 YouTube video search
* 🖼️ Thumbnail extractor
* 📊 Video analyzer
* 🧠 AI-based summary generation
* 🔐 User authentication (Login / Signup UI)
* 🌐 Fully responsive UI

---

## 🛠️ Tech Stack

### Backend

* Java (Spring Boot)
* REST APIs

### Frontend

* HTML
* CSS
* JavaScript

### APIs Used

* Google Gemini API (AI content generation)
* YouTube Data API v3

### Deployment

* Docker
* Render
